# Wine Price Prediction Dashboard

Wine price prediction using statistical models and machine learning (GAM, Random Forest) – academic project developed at Universitat Pompeu Fabra.

## Key Results
- Best model: Random Forest  
- Performance: R² ≈ 0.76, RMSE ≈ 0.34  

## Project Overview
The goal of this project is to predict wine prices using chemical and commercial characteristics such as alcohol by volume (ABV), vintage, grape variety, wine style, closure type, and country of origin.

## Methodology
- Data cleaning and preprocessing  
- Exploratory Data Analysis (EDA)  
- Feature engineering  
- Model comparison  

## Models
- Linear Regression  
- Generalized Additive Models (GAM)  
- Random Forest
